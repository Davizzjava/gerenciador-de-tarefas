# ✅ Premium Task Manager

A modern, feature-rich To-Do application built with vanilla JavaScript. It demonstrates state management, DOM manipulation, and data persistence using the browser's Local Storage.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure.
* **CSS3:** Modern UI, gradients, and interactive states.
* **JavaScript (Vanilla):** Array methods, Object handling, and Event Listeners.
* **Local Storage:** Data persistence across page reloads.

## 💡 Features
* Add, remove, and toggle tasks as completed.
* **Data Persistence:** Tasks are safely stored in the browser.
* Dynamic pending tasks counter.
* Friendly "Empty state" handling for better UX.
